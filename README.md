### Сборка
```bash
dotnet build
```

### Запуск с параметрами
```bash
# Узел 1 - Администратор
dotnet run -- -u Damir -p 5000 -a

# Узел 2 - Пользователь (подключается к админу)
dotnet run -- -u Timur -p 5001 -c 127.0.0.1:5000

# Узел 3 - Ещё один пользователь
dotnet run -- -u Ayrat -p 5002 -c 127.0.0.1:5000
```

### Интерактивный режим
```bash
dotnet run
# Приложение запросит имя, порт и роль
```

## 💬 Команды чата

| Команда | Описание |
|---------|----------|
| `/connect <host:port>` | Подключиться к другому узлу |
| `/users` | Показать список онлайн |
| `/pm <user> <msg>` | Приватное P2P-зашифрованное сообщение |
| `/emojis` | Показать доступные смайлики |
| `/me` | Информация о своём профиле |
| `/clear` | Очистить экран |
| `/quit` | Выход |

### Команды администратора
| Команда | Описание |
|---------|----------|
| `/ban <user>` | Забанить пользователя |
| `/unban <user>` | Разбанить пользователя |
| `/banlist` | Показать бан-лист |

### Broadcast-шифрование
- Все сообщения в общий чат шифруются **AES-256**
- Общий ключ генерируется администратором и распространяется при подключении

### P2P-шифрование (приватные сообщения)
1. При подключении узлы обмениваются публичными ключами **RSA-2048**
2. Для каждого приватного сообщения генерируется случайный **AES-256 ключ**
3. Сообщение шифруется AES, ключ шифруется RSA публичным ключом получателя
4. Только получатель может расшифровать сообщение своим приватным ключом
